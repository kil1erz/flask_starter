# Flask Starter

最小可运行的 Flask 后端模板，已配置 VS Code 调试。

## 快速开始
```bash
python -m venv venv
source venv/bin/activate   # 激活Python虚拟环境
pip install -r requirements.txt
python app.py
```

## 常用Git命令

#### 查看状态
git status

#### 查看提交历史
git log

#### 拉取最新更改
git pull

### 修改文件

#### 检查状态

#### 提交更改信息
